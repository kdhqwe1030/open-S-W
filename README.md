
# 🐕 with_pet: 반려동물과 함께 추억을 만들어가는 서비스

<br/>

## ✨ 주요 기능

### 1️⃣ 반려동물과 함께 한 산책 경로 기록

&nbsp;산책을 시작하면 사용자가 움직이는 경로를 기록합니다!

### 2️⃣ 반려동물 등록 및 정보 기록

&nbsp;

### 3️⃣ 캘린더를 통한 추억 기록

&nbsp; 캘린더를 통해 여러가지 추억을 기록할 수 있습니다!
